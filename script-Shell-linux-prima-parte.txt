esercizio 3.

scrivere uno script che controlla se esiste un file /etc/shadow. Se il file esiste allora visualizza il messaggio "shadow password are enabled" Poi bisogna controllare se è possibile scrivere il file se così "you have permissions to edit /etc/shadow" se non puoi, allora visualizza il messaggio "you do not have permissions to edit /etc/shadow"




#!/bin/bash

PERCORSO="/etc/shadow"

if [ -e $PERCORSO ] then

   echo " ${PERCORSO} shadow password are enabled"

   if [ -w $PERCORSO ] then

	echo "you have permissions to edit ${PERCORSO} "

   else
	echo "you do NOT have permissions to edit ${PERCORSO} "

   fi


fi
